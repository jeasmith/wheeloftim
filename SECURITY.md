# Security Policy

## Supported Versions

We actively support the latest version of the application.

## Reporting a Vulnerability

If you find a security vulnerability, please report it by:

1. Creating a security advisory in the GitHub repository
2. Or emailing the maintainers directly

Please do not create a public issue for security vulnerabilities.

## Security Features

This project uses:
- GitHub Advanced Security (GHAS) with CodeQL scanning
- Automated dependency updates via Dependabot
- Regular security audits
